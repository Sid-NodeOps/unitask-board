# Minimal Task Board

Una board per il task management semplice, veloce e minimalista per gestire le tue attività quotidiane.

## Caratteristiche

- **Design Minimalista**: Interfaccia pulita e senza distrazioni.
- **Kanban Board**: Tre colonne (Da Fare, In Corso, Completate) con supporto Drag & Drop.
- **Gestione Priorità**: Livelli di priorità (Alta, Media, Bassa) con indicatori visivi.
- **Commenti**: Aggiungi note e aggiornamenti ad ogni task.
- **Date Scadenza**: Tracciamento delle scadenze con evidenziazione per i ritardi.
- **Salvataggio Locale**: I dati vengono salvati automaticamente nel browser.

## Tecnologia

- HTML5
- Vanilla JavaScript
- Tailwind CSS (via CDN)
- FontAwesome Icons

## Utilizzo

1. Apri `index.html` nel tuo browser.
2. Clicca su "+ Nuova Task" per iniziare.
3. Trascina le card tra le colonne per aggiornare lo stato.
4. Clicca su una card per vedere i dettagli, modificare o aggiungere commenti.
